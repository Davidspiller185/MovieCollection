# MovieCollection
**תיאור הפרויקט**
```text
מטרת הפרויקט היא לבנות מערכת שאוספת סרטים ומאפשרת למשתמש לבמצע פעולות של הוספת סרט או מחיקת סרט וכדומה באמצעות התפריט.
```

**מבנה הפרויקט**
```text
movie-manager/
│
├── app.js
├── package.json
│
├── data/
│ └── movies.json

│
├── services/
│ ├── file.service.js
│ └── movie.service.js
│
└── utils/
└── validator.js
```

**התקנות**
```bach
npm init -y
npm install readline-sync
```
